Follow these simple steps to get started on Windows:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1: Install Node.js
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: https://nodejs.org/
2. Download the "LTS" version (left button)
3. Run the installer (.msi file)
4. Click "Next" through all steps (keep defaults)
5. Restart your computer

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2: Install MongoDB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OPTION A - Local MongoDB (Recommended for learning):
1. Go to: https://www.mongodb.com/try/download/community
2. Select: Windows, Latest version
3. Download and run the installer
4. Choose "Complete" installation
5. Check "Install MongoDB as a Service"
6. Finish installation

OPTION B - MongoDB Atlas (Cloud - Easier but needs internet):
1. Go to: https://www.mongodb.com/cloud/atlas
2. Sign up for FREE account
3. Create a FREE cluster
4. Get your connection string
5. Use this in Step 4 below

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3: Get Groq API Key (CRITICAL FOR AI FEATURES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WITHOUT this key, Chat, Summary, Flashcards, and Quiz will NOT work!

1. Go to: https://console.groq.com/keys
2. Sign up for FREE account (free API calls!)
3. Click "Create API Key"
4. COPY the key (starts with gsk_...)
5. SAVE IT SOMEWHERE SAFE
6. You'll add this in Step 5

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4: Setup the Project
